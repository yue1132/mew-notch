# MewNotch 使用说明

## 简介

MewNotch 是一款免费开源的 macOS 应用，将 Mac 凹口（刘海）转换为可自定义的状态显示区。支持亮度、音量、电源状态、媒体播放等功能显示，让凹口真正变得实用！

## 功能特性

### 基础 HUD 功能

#### 亮度显示
- 实时显示亮度调节变化
- 支持自定义步长大小
- 可选显示自动亮度变化

#### 音量显示
- 显示输入/输出音量变化
- 直接在凹口上显示
- 可自定义步长大小

#### 电源状态
- 显示当前电源来源（电池/电源适配器）
- 可选显示剩余时间
- 锁屏状态下也可见

#### 媒体播放控制
- 直接从凹口控制正在播放的媒体
- 自动显示标题变化
- 支持播放/暂停控制
- 悬停显示应用图标，点击打开应用

#### 镜像视图
- 在展开凹口中快速查看自己的样子
- 可自定义圆角半径

#### Bash 命令视图
- 在展开凹口中运行和显示 Bash 命令输出

#### 文件收纳架
- 拖放文件到凹口快速访问
- 重启后文件仍然保留
- 支持文件分组管理

### 生产力工具（v2.3.0 新增）

#### 🍅 番茄钟
专注计时器，使用番茄工作法提高效率：

**折叠状态显示**：
- 凹口左侧：状态图标（工作🧠、短休息☕、长休息🚶）
- 凹口右侧：倒计时时间
- 实时刷新，每秒更新

**展开状态功能**：
- 完整番茄钟控制界面
- 开始/暂停/重置/跳过
- 工作时长：25 分钟（可自定义）
- 短休息：5 分钟（可自定义）
- 长休息：15 分钟（可自定义）
- 显示已完成番茄数

**使用方法**：
1. 点击番茄钟 MiniView 进入详细视图
2. 点击播放按钮开始工作计时
3. 计时结束后自动切换到休息
4. 4 个番茄后自动进入长休息

#### 📅 日历集成
显示今日日程和即将到来的事件：

**功能**：
- 显示今日所有事件
- 高亮显示正在进行的事件
- 显示下一个即将到来的事件
- 显示事件时间和地点
- 支持刷新事件列表

**权限要求**：
首次使用需要授权日历访问权限：
1. 点击日历视图
2. 点击"授权访问"按钮
3. 在系统弹窗中允许日历访问

#### 📝 提词器
导入脚本进行提词演示：

**功能**：
- 导入文本文件作为脚本
- 支持滚动播放
- 可调节滚动速度
- 显示当前进度

**使用方法**：
1. 点击提词器 MiniView 进入详细视图
2. 点击"导入脚本"选择文本文件
3. 点击播放开始滚动
4. 悬停显示控制按钮

#### ✅ 待办事项提醒
跟踪任务完成情况：

**功能**：
- 显示待办事项列表
- 过期任务红色高亮提醒
- 显示待办数量徽章
- 与系统提醒事项集成

**权限要求**：
需要授权提醒事项访问权限

#### ⏱️ 计时器
自定义倒计时计时器：

**功能**：
- 创建多个计时器
- 自定义计时时长
- 显示进度条
- 计时结束提醒

### 多语言支持（v2.3.0 新增）

- 完整的中英文界面切换

**切换方法**：
设置 → 通用 → 语言 → 选择中文/英文

## 使用方法

### 基础操作

1. **启动应用**：双击 MewNotch.app 启动
2. **打开设置**：再次点击应用图标或通过菜单栏图标
3. **触发显示**：使用键盘调节亮度/音量，凹口会显示变化
4. **展开凹口**：鼠标悬停在凹口上，自动展开详细视图
5. **锁定展开**：点击锁定按钮，凹口保持展开状态

### 自定义设置

#### 凹口设置
- **显示位置**：选择在哪些显示器显示凹口
  - 仅刘海显示器
  - 所有显示器
  - 指定显示器
- **悬停延迟**：调整悬停展开的延迟时间
- **圆角半径**：自定义凹口圆角大小
- **玻璃效果**：开启/关闭背景玻璃效果

#### HUD 设置
- **样式选择**：
  - Minimal（极简）
  - Progress（进度条）
  - Notched（凹口样式）
- **动画效果**：开启/关闭 HUD 动画
- **系统 HUD**：隐藏 macOS 默认 HUD

#### 展开项设置
- **布局排序**：拖拽调整展开项顺序
- **启用/禁用**：勾选需要显示的功能
- **分隔线**：显示/隐藏项目分隔线

### 折叠状态 HUD

折叠状态下，凹口显示以下信息（左右分布）：

**左侧 HUD**：
- 亮度变化
- 输入音量（麦克风）
- 输出音量（扬声器）
- 番茄钟状态图标（运行时）

**右侧 HUD**：
- 输出音量（扬声器）
- 输入音量（麦克风）
- 亮度变化
- 番茄钟倒计时（运行时）

### 展开状态视图

展开后显示：
- **顶部**：标签切换器、设置按钮、锁定按钮
- **主页**：MiniView 小部件（番茄钟、日历、提词器、待办、计时器、音乐播放等）
- **详细视图**：点击 MiniView 进入详细界面

## 安装方法

### Homebrew 安装

```bash
brew install --cask monuk7735/tap/mew-notch
```

### 手动下载安装

1. 从 [GitHub Releases](https://github.com/monuk7735/mew-notch/releases) 下载最新版本
2. 将应用移动到 Applications 文件夹
3. 运行应用并授予必要权限

### ⚠️ "已损坏"或"未识别开发者"错误解决

由于开发者没有 Apple Developer 账号，首次启动会显示警告。

**方法 1（推荐）：通过系统设置允许**

1. 打开 **系统设置** → **隐私与安全性**
2. 滚动到 **安全性** 部分
3. 找到 "MewNotch was blocked..."，点击 **仍要打开**
4. 在确认弹窗中点击 **打开**

**方法 2（高级）：终端命令**

```bash
xattr -cr /Applications/MewNotch.app
```

此命令移除 macOS 对网络下载应用的"隔离"标记。

## 权限要求

### 必需权限

1. **辅助功能权限**：用于监听媒体键（亮度/音量调节键）
   - 系统设置 → 隐私与安全性 → 辅助功能
   - 添加 MewNotch 到允许列表

2. **麦克风权限**（可选）：用于显示输入音量变化
   - 首次调节麦克风音量时会请求

### 生产力工具权限

3. **日历访问权限**：用于日历功能
   - 点击日历视图时请求

4. **提醒事项权限**：用于待办事项功能
   - 点击待办视图时请求

## 常见问题

### Q: 凹口没有显示？
A: 
- 检查应用是否正在运行（菜单栏图标）
- 非刘海显示器需要手动启用（设置 → 凹口 → 显示位置）
- 尝试调节亮度或音量触发显示

### Q: 番茄钟倒计时不更新？
A:
- 确保番茄钟已开始（点击播放按钮）
- 折叠状态会实时显示在凹口两侧
- 展开状态查看详细视图

### Q: 展开视图超出屏幕？
A: v2.3.0 已修复，展开视图限制最大宽度不超过屏幕

### Q: 如何隐藏系统默认 HUD？
A: 设置 → 折叠项 → 各 HUD 设置 → 隐藏系统 HUD

### Q: 如何切换语言？
A: 设置 → 通用 → 语言 → 选择中文或英文

### Q: 如何更新应用？
A: 
- 应用内置 Sparkle 自动更新
- 设置 → 关于 → 点击"检查更新"
- 也可手动下载新版本
